with HPE OneView Rest API version 300 or higher. Initial release supports
the following features:

 - Service Root
 - System collection
 - System
 - System (Boot, Processor and Memory mappings)
 - Chassis collection
 - Chassis
 - Thermal mappings for Chassis
 - Manager collection
 - Manager
 - Error responses
